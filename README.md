A shell/command-line interpreter with the following capabilities:
Reads lines of user input, parses them and executes the commands by forking off new processes;
Executing commands in the background using the & symbol; 
Redirecting the standard input, standard output and standard error streams to or from files using the <, > and 2> symbols; 
Redirecting and appending the standard output and standard error streams to files using the >> and 2>> symbols;
Piping the standard output of one process to the standard input of another using the | symbol;
